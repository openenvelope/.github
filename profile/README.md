# Envelope

Envelope is the multi-agent designer. Describe what you need in plain language — Envelope structures it into a complete sub-agent design with named agents, roles, model assignments, tool access, and handoff logic, then exports it as a schema-valid `.envelope.json` file.

## What's here

- **[schema](https://github.com/openenvelope/schema)** — The open JSON Schema for multi-agent team definitions. Apache 2.0. Any runtime can implement it.
- **[claude-plugin](https://github.com/openenvelope/claude-plugin)** — Claude Code skill for designing multi-agent workflows directly from your terminal.
